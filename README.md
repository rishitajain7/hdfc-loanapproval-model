Loan Approval Prediction Model

📌 Project Overview
This project predicts whether a loan will be approved based on applicant data.

📁 Files
- hdfc_train.py → training model
- hdfc_predict.py → prediction script
- loan_model.pkl → trained model
- scaler.pkl → scaling object
- imputer.pkl → missing value handler

⚙️ How to Run
1. Install requirements
2. Run training script:
   python hdfc_train.py
3. Run prediction:
   python hdfc_predict.py

📊 Dataset
- loan_approval_dataset.csv
